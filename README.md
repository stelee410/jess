# jess
An AI girl friend in memory of my broken love

## 设置虚拟环境

1. 创建虚拟环境：
   ```bash
   python -m venv .venv
   ```

2. 激活虚拟环境：
   - Windows:
     ```bash
     .venv\Scripts\activate
     ```
   - macOS和Linux:
     ```bash
     source .venv/bin/activate
     ```

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

   或者直接安装`openai`：
   ```bash
   pip install openai
   ```
